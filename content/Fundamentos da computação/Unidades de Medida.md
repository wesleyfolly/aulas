# Unidades de Medida

1. **Introdução:**
    - O que são unidades de medida na computação?
2. **Bit e Byte:**
    - Definição de Bit e Byte.
    - Como eles são usados para representar informação.
3. **Unidades de Medida para Armazenamento de Dados:**
    - Kilobyte (KB), Megabyte (MB), Gigabyte (GB), Terabyte (TB), Petabyte (PB).
    - Explicação da progressão exponencial.
    - Diferença entre a definição de base 10 (usada em dispositivos de armazenamento) e base 2 (usada em memória de computador): KiB, MiB, GiB, TiB.
4. **Unidades de Medida para Velocidade de Dados:**
    - Kilobit por segundo (Kbps), Megabit por segundo (Mbps), Gigabit por segundo (Gbps), etc.
    - Como elas são usadas para medir velocidade de internet, transferência de dados, etc.
5. **Unidades de Medida para Processamento de Dados:**
    - Hertz (Hz), Kilohertz (KHz), Megahertz (MHz), Gigahertz (GHz), Terahertz (THz).
    - Como elas são usadas para medir a velocidade de processamento da CPU.
6. **Conversões de Unidades:**
    - Como converter entre diferentes unidades de medida.
    - Prática de problemas de conversão.
7. **Conclusão:**
    - A importância das unidades de medida na computação.
    - Como elas afetam o desempenho do computador e o armazenamento de dados.
