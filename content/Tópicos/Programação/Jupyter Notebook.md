# Jupyter Notebook

## **O que são Notebooks?**

Um Notebook se parece com um **caderno**, onde será escrito uma história. Essa história é narrada em partes e segue um fluxo lógico, desde a introdução até a conclusão. 

É uma forma de ter blocos de texto (contexto) e código juntos no mesmo lugar.

O **Jupyter Notebook** é um ambiente que oferece essa abordagem de Notebooks, com um visual simples e muito fácil de utilizar.

# Opções de instalação

1. Instalar o pacote Anaconda (que vem com muitos outros recursos)
2. pip install jupyter (já precisa ter o python instalado)
3. instalar um plug-in no VSCode
4. Online - [https://jupyter.org/try](https://jupyter.org/try)
