# Tipos de ataques

1. **Ataque de Phishing**: Enganar usuários para revelar informações pessoais.
2. **Ataque de Força Bruta**: Tentativas repetidas de adivinhar credenciais.
3. **Ataque de Malware**: Uso de software malicioso para acessar sistemas.
4. **Ataque Man-in-the-Middle (MitM)**: Interceptação de comunicações entre duas partes.
5. **Ataque de Negação de Serviço (DoS)**: Sobrecarga de um sistema para torná-lo inacessível.
6. **Ataque de SQL Injection**: Injeção de código SQL malicioso em um banco de dados.
7. **Ataque de Cross-Site Scripting (XSS)**: Injeção de scripts maliciosos em sites da web.
8. **Ataque de Zero Day**: Exploração de vulnerabilidades desconhecidas.
9. **Ataque de Eavesdropping**: Escuta indesejada de comunicações privadas.
10. **Ataque de Pass-the-Hash**: Uso indevido de hashes de senha para autenticação.
11. **Ataque de Spoofing**: Falsificação de origem, como endereços IP ou e-mails.
12. **Ataque de Session Hijacking**: Tomada de controle de uma sessão de usuário.
13. **Ataque de Social Engineering**: Manipulação psicológica para obter informações confidenciais.
14. **Ataque de Typosquatting**: Uso de URLs similares a populares para enganar usuários.
15. **Ataque de Pharming**: Redirecionamento de tráfego legítimo para sites fraudulentos.
