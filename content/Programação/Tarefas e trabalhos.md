# Tarefas e trabalhos

- Apresentação de trabalho sobre Paradigmas de programação
    1. Para cada equipe será sorteado um paradigma.
    2. O trabalho deve ter teoria e prática rodando um código.
    3. A turma deve participar da atividade prática.
    4. A turma deve ser **avaliada**. Como? Através de um Quiz. —> https://quizizz.com/?lng=pt-BR
    5. Modelo de apresentação é livre.
    6. Qual linguagem devo utilizar? A que a equipe definir.
    7. Pode ser utilizado o Replit como framework https://replit.com/
    
- Realizar os desafios práticos da linguagem escolhida
    
    O professor fará aulas práticas de programação em python
    
    O aluno ou equipe escolherá uma outra linguagem de sua preferência para realizar os desafios.
    
- Apresentação de trabalho sobre novas tecnologias e inovações na área da programação
    
    Cada equipe ou aluno escolherá um tema, que não pode ser repetido
    
    Podemos utilizar a lista da página [[Novas tecnologias e pesquisa]]
